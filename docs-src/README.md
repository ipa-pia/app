# 智能个人助手

基于tauri开发的一款桌面app，插件式机制，允许用户自定义插件

支持平台：

- [x] windows
- [ ] linux
- [ ] macos


## 功能特性
1. 自定义插件、插件商店
2. 明亮、暗黑主题
3. 内置：剪贴板、TODO、翻译、Markdown、JSON等功能
4. 更多功能积极开发中。。。
